# Currículo - Rafael dos Reis Cardoso

Este é um projeto pessoal para apresentar o meu currículo em formato de site. O objetivo é demonstrar minhas habilidades e experiências, além de praticar desenvolvimento Front-End.

## 🖥️ Demonstração

Você pode acessar o site publicado aqui: [Currículo Online](https://github.com/CardosoReis/curriculo-online.git)  

---

## 📂 Estrutura do Projeto

- **index.html**: Página inicial com informações sobre mim.
- **style/style.css**: Arquivo de estilo utilizado para customizar o layout.
- **imagens/**: Pasta contendo imagens do projeto, como fotos de perfil e ícones.
- **CV_Rafael_dos_Reis_Cardoso.pdf**: Arquivo do currículo para download.

---

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura do site.
- **CSS3**: Estilização do layout.
- **GitHub Pages**: Hospedagem do site.

---

## 📜 Funcionalidades

- **Apresentação Pessoal**: Informações sobre minha trajetória profissional e habilidades.
- **Educação e Experiência**: Páginas dedicadas às minhas formações acadêmicas e experiências profissionais.
- **Contato**: Links diretos para e-mail e WhatsApp.
- **Download do Currículo**: Botão para baixar o currículo em formato PDF.
